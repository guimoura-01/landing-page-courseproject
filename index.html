Página Grid

Esta é a pagina na qual eu trabalho com um "display: grid".
Eu fiz diversas manipulações para que ficasse da melhor forma visualmente.

Resumo

A grid foi criada dentro de uma tag section, a qual contém todo o conteúdo da seção da página. A grid em questão possui 3 colunas e cada uma utiliza 1fr (uma fração). Os números apresentados antes dos títulos foram criados através de um contador e apresentados na forma de um pseudo-elemento.

Summary

The grid was created inside a section tag, which contains all the content of the section of the page. The grid in question has 3 columns and each uses 1fr (a fraction). The numbers displayed before the titles were created using a counter and presented in the form of a pseudo-element.

Resumen

La rejilla se creó dentro de una etiqueta section, que contiene todo el contenido de la sección de la página. La rejilla en cuestión tiene 3 columnas y cada una utiliza 1fr (una fracción). Los números que aparecen antes de los títulos se crearon utilizando un contador y se presentaron en forma de pseudoelemento.
